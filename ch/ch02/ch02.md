## 线性代数

### P23

```
向量的乘以一个系数只会延长或缩短距离, 不会改变在N维空间中的方向。
假设A是一个3x2的矩阵, 所以A可看成两个列向量A:1和A:2的列向量组
存在列向量[x1,x2]T, x1,x2可为任意常数。 
而 A:1*x1和A:2*x2 只能描述一个平面, 这个平面可能是三维空间中的一个切面。
无论如何改变[x1,x2]T 的值都只能表示R^2空间。

简单解释：A:1 为一条线, A:2 为另一条线，两条线只能组成一个平面。
```

### P27

```
实对称矩阵的不同特征值对应的特征向量相互正交
证明:
设: 
①  Ax = ax
②  Ay = by
则① 两边同时乘以yT 有:  ④ yTAx = yTax,
则② 两边同时乘以xT 有:  ⑤ xTAy = xTby,

因为:
A是实对称矩阵，则
yTAx == xTAy ⑥ 
且
xTy == yTx ⑦ 

所以
④  - ⑤  有: 0 = (a - b)xTy

因为特征值 a!=b
所以xTy == 0
所以x y 为正交向量。

ps: ④ 证明:
因为x, y为列向量, xT和yT为行向量
所以xTAy和yTAx都是一个实数
一个实数的转质等于自身
xTAy == (xTAy)T == xAyT
```
